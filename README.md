# Local Community App

The Local Community App provides a platform for users to connect with economic opportunities and community services.

## Features
1. Fundraising campaigns
2. Job opportunities
3. Marketplace services
4. Community effect analytics

## Purpose
Transform economic empowerment globally through a Flutter-based platform connecting fundraising campaigns, job opportunities, marketplace services, and community effect analytics.

## User Guide
- **How to Use**: Register and start connecting.
- **Feedback**: Provide feedback through our contact form.

## Community Effect Statistics
Community Effect Statistics: Calculate per-capita contribution effect with population data.
